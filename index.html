Review Materi Praktikum SIGWEB

Praktikum Sistem Informasi Geografis merupakan salah satu mata kuliah yang menjelaskan terkait bagaimana cara menyajikan informasi geospasial dalam sebuah platform yang dapat diakses oleh semua orang. Pada mata kuliah ini diajarkan terkait HTML (Hypertext Markup Language), CSS (Cascading Style Sheet), dan JS (Java Script). HTML merupakan bahasa markup, yaitu bahasa yang menunjukkan sebagai objek. HTML ini berperan sebagai motor utama yang merupakan objeknya. Di dalam HTML biasanya dapat ditemui data vektor dan data raster. Selanjutnya CSS sebagai pengaturan style. Dalam file CSS dilakukan pengaturan style pada tampilan website yang dirancang. CSS bisa dibuat dalam file terpisah yang kemudian akan dipanggil dalam file HTML atau didefinisikan langsung pada file HTML yang disebut dengan pendefinisian in line. Kemudian JS merupakan file yang bertugas sebagai pengaturan aplikasi misalnya seperti pop up atau lainnya sehingga website tampak lebih interaktif.

Dalam mata kuliah ini juga diajarkan terkait dengan GitHub. GitHub merupakan layanan repository Git dengan berbagai fungsionalitasnya ditambah dengan fungsi-fungsi lain yang memperkaya manajemen kode pada Github. Dengan menggunakan GitHub, kita dapat mengupload hasil coding dari website yang kita buat. Selain itu, kita juga dapat berpartisipasi dalam codingan yang dibuat oleh orang lain dengan menggunakan mekanisme Forking. Mekanisme forking merupakan salah satu fungsi GitHub yang mana kita dapat mengambil template dari developer lain untuk kemudian dilakukan pembaruan atau penambahan fungsi sehingga tidak perlu membuat halaman web dari awal. Mekanisme ini menggunakan aplikasi Ubuntu dan Visual Studio Code untuk melakukan cloning, update, dan pembaruan. Untuk lebih jelasnya, dapat melihat tutorial yang ada pada link berikut Tutorial GitHub.
